# Arduino-wifi-waves
基于Arduino的超声波测距项目  
---  
* 设计思路：  
使用超声波和舵机，探测物体的方向和距离，并使用温湿度模块修正数据。后期可能加入wifi模块进行数据传输。
---  
* 所需硬件  
1.USB转TTL  
2.5v继电器  
3.3个led彩灯  
4.SG90  9g舵机  
5.杜邦线  
6.usb线  
7.DHT11 温度湿度模块  
8.HC-SR04超声波模块  
9.面包板+电源模块  
10.ESP-8266 wifi模块  
11.arduino uno R3板  
12.220欧姆电阻    
13.公母线  
14.1602A l2c显示屏  
15.DC电源线
---  
* 软件平台  
Arduino IDE  
Arduino Box  
windows10  /Ubuntu 17.10
